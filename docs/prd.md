# Product — Momento

## Vision
Momento is a **shared moments hub**.  
A single place to organize an event and centralize its photos afterward.

Events are the means. Memories are the core value.

---

## Problem

Event photos are currently scattered:
- Messaging apps compress images
- Cloud galleries mix personal and shared photos
- Links get lost
- Not everyone has access
- Hard to revisit later

There is no simple place where **event + participants + photos** coexist.

---

## Product Goal
Allow any group to capture and relive shared moments in a private space.

Expected outcomes:
- Photos centralized in one place
- Clear participant boundaries
- Easy revisiting of memories

---

## Target Users

People organizing shared moments:
- friends
- families
- couples
- trips
- birthdays
- parties
- large or small gatherings

No size limit.

---

## Value Proposition

Everything about a moment in one place:  
event, people, and private photos.

---

## Core Product Principles

- Events are private by default  
- Only invited users can access photos  
- Photos belong to an event  
- Events naturally transition into memories over time  

---

## Core User Journey

1. User creates an event  
2. Invites participants  
3. The event happens  
4. Participants upload photos  
5. Event becomes a shared memory  
6. Users revisit the gallery later
